# fluent-plugin-jwt

[Fluentd](https://fluentd.org/) filter plugin to sign and decode JSON Web Token (JWT).

## Installation

### RubyGems

```
$ gem install fluent-plugin-jwt
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-jwt"
```

And then execute:

```
$ bundle
```
